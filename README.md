# NonPolitica-Game
Управляйте государством
УПРАВЛЕНИЕ ГОСУДАРСТВОМ - ТЕКСТОВАЯ СТРАТЕГИЯ

ОПИСАНИЕ

Вы становитесь правителем государства. Вам предстоит управлять страной, принимать решения и довести её до процветания за 100 игровых шагов.

ОСОБЕННОСТИ

- 100 шагов управления
- Случайные события с выбором действий
- 5 слотов для сохранения игры
- Более 15 параметров состояния
- Экологическая система с пороговыми эффектами
- 7 условий поражения

ИГРОВЫЕ ПАРАМЕТРЫ (начальные значения)

Население: 100000 человек
Бюджет: 10000000000
Стабильность: 40%
Доверие: 75%
ВВП: 1
Еда: 1000000
Вода: 500000
Промышленность: 3
Инфраструктура: 0
Загрязнение: 1
Конвои: 10
Грузовики: 50
Вертолёты: 1
Оружие: 250
Боеприпасы: 10000
Тяжёлое вооружение: 50
Парламент: 15

УСЛОВИЯ ПОРАЖЕНИЯ

Игра заканчивается если:
1. Стабильность падает до 0 - государство распадается
2. Доверие падает до 0 - начинается революция
3. Еда заканчивается - голод
4. Вода заканчивается - засуха
5. Население сокращается до 1000 человек
6. Загрязнение достигает 100 - экологическая катастрофа
7. Бюджет становится отрицательным - банкротство

СИСТЕМА СОБЫТИЙ

Каждый шаг генерируется случайное событие из 6 категорий:

1-150 (15%): События с едой и водой
151-400 (25%): Промышленные события
401-450 (5%): Инфраструктурные события
451-475 (2.5%): Военные события
476-750 (27.5%): Демографические события
751-1000 (25%): Политические и экономические события

У каждого события есть 3-5 вариантов действий с разными последствиями.

СИСТЕМА ЗАГРЯЗНЕНИЯ

При достижении определённых порогов применяются штрафы:

Загрязнение 20+: Стабильность -5%, Еда -20000, Вода -10000
Загрязнение 40+: Стабильность -8%, Еда -40000, Вода -20000
Загрязнение 60+: Стабильность -12%, Еда -60000, Вода -30000
Загрязнение 80+: Стабильность -15%, Еда -80000, Вода -40000
Загрязнение 100: Мгновенное поражение

СОХРАНЕНИЕ ИГРЫ

Игра поддерживает 5 слотов для сохранения. Все сохранения хранятся в папке "saves" рядом с файлом игры. В меню сохранений доступны: сохранение, загрузка, удаление, просмотр всех сохранений и открытие папки.
